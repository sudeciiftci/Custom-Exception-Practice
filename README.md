# Java Custom Exception Practice

This project demonstrates how to create and use **custom exceptions** in Java.

## Concepts
- Creating custom exception classes
- Extending `Exception`
- Throwing custom exceptions
- Handling exceptions with `try-catch`

## Examples

### CustomEx1
- Creates `InvalidAgeException`.
- Throws an exception for an invalid age.

### CustomEx2
- Creates:
  - `InvalidAmountException`
  - `InsufficientBalanceException`
- Simulates an ATM withdrawal with custom error handling.

## Purpose
- Understand custom exceptions
- Practice validation and error handling
- Improve program reliability

## Run
Run `CustomEx1` and `CustomEx2` separately.
